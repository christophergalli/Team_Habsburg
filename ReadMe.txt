This repository was created for the module "Medical Software Development" of the MSc Medical Informatics. 

Owners are: Christopher Galli, Jan Kruta, Urs Känel, Amun Stalder.

The repository contains the following artifacts:
1) genedataloader (data loader component)
2) generestservice (rest service component)
3) genewebapp (web application component)

In addition, we have uploaded the following documents:
1) Presentation of the project (.ppt)
2) Documentation for the client (.docx)



Note: If you are not a windows user, please modify the location of the log file for the webapp as follows:
-	Logback.xml (within the resource folder)
-	Line 4: value=”path”
